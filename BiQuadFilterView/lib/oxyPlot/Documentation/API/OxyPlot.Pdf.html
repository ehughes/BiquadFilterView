OxyPlot.Pdf Namespace

The OxyPlot.Pdf namespace contains functionality to export plots and reports to .pdf files.

Classes

Name Description
LatexReportWriter Provides a report writer for LaTeX2e.
PdfExporter Provides functionality to export plots to pdf.
PdfReportWriter Provides a report writer for portable document format using MigraDoc.
PdfSharpExtensions Provides extension methods for OxyPlot to PdfSharp type conversion.
RtfReportWriter Provides a report writer for rich text format using MigraDoc.
